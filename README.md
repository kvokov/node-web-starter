## Getting started

```shell
$ git clone https://github.com/kvokov/node-web-starter.git my-app
$ cd my-app
$ npm install
$ npm start
```
**That's it!** :sunglasses:

## NPM commands

`$ npm start` - Run in development mode

`$ npm test` - Run automated tests

`$ npm run build` - Make production build
